Tentang Saya

Halo namaku Michael Isaiah. Lahir tanggal 20 Maret 2002. Sekarang saya tinggal di Cicaheum, Bandung. Saya sedang berkuliah di UNIKOM. Hobi saya main game, bulu tangkis, berenang, dan traveling. Saya juga senang dengan hewan, di rumah pelihara berbagai hewan, ada anjing, kura-kura, ikan, burung, dan ayam.

Saya suka sekali dengan proses mengkoding dan juga mencoba hal yang belum pernah saya coba sebelumnya. Kalau ada tugas mengkoding saya ingin langsung mengerjakannya. Saya berharap anda bisa memberi saya kesempatan untuk bergabung atau komunitas anda. Terima kasih. Semoga harimu menyenangkan.
